# FileChecker

v2 DD 230215Cancel changes

#### 기능
1. 파일의 모든 메모(노트) 일괄 삭제
2. 파일의 모든 오류난 함수 일괄 삭제

#### 함수
IsThere(arg1, arg2)  
arg1에 특정 셀, arg2에 특정 범위(열)을 넣으면 arg1이 arg2에 존재하는지 여부를 True/False로 반환

ex.
=IsThere(A1, B1:B10)

#### 사용방법
1 xlam파일을 다운  
2. 탐색기에서 우클릭하여 "일반" 보안: 의 "차단 해제" 클릭  
3. 필요한 때 실행하면 엑셀 상단에 탭 생성됨
